# API Reference

Full interactive OpenAPI documentation is auto-generated by FastAPI and
served at `/api/docs` (Swagger UI) and `/api/redoc` (ReDoc) once the backend
is running â€” this file is a high-level index, not a substitute for it.

Base path: `/api/v1`. All endpoints except `/auth/login` and `/health`
require `Authorization: Bearer <access_token>`.

## Auth
| Method | Path | Role | Description |
|---|---|---|---|
| POST | `/auth/login` | â€” | Exchange username/password for JWT access + refresh tokens |
| POST | `/auth/change-password` | any | Change your own password |

## Users
| Method | Path | Role | Description |
|---|---|---|---|
| GET | `/users` | lecturer+ | List users, optional `?role=` filter |
| POST | `/users` | lecturer+ | Create a user (lecturer can't create admins) |
| POST | `/users/{id}/reset-password` | lecturer+ | Reset a user's password |
| POST | `/users/{id}/disable` | lecturer+ | Disable an account |
| POST | `/users/{id}/enable` | lecturer+ | Re-enable an account |

## Operating Systems (catalogue)
| Method | Path | Role | Description |
|---|---|---|---|
| GET | `/operating-systems` | any | List the OS catalogue |
| POST | `/operating-systems/rescan` | lecturer+ | Force a re-sync from `VMImages/` |

## VMs
| Method | Path | Role | Description |
|---|---|---|---|
| POST | `/vms` | any | Launch a VM (`{ "os_slug": "..." }`); `202 Accepted` |
| GET | `/vms/mine` | any | Your own active session, if any |
| GET | `/vms` | lecturer+ | All active sessions across all students |
| POST | `/vms/{id}/stop` | owner | Stop your own VM |
| POST | `/vms/{id}/force-stop` | lecturer+ | Force-stop any VM |
| POST | `/vms/{id}/extend` | owner | Extend session (once, up to policy max) |
| POST | `/vms/{id}/activity` | owner | Heartbeat for idle-timeout detection |

## Admin
| Method | Path | Role | Description |
|---|---|---|---|
| GET | `/admin/host-stats` | lecturer+ | Live host CPU/RAM + VM counts |
| POST | `/admin/maintenance/enable` | lecturer+ | Broadcast maintenance mode |
| POST | `/admin/maintenance/disable` | lecturer+ | Clear maintenance mode |
| GET | `/admin/audit-logs` | lecturer+ | Recent audit trail entries |

## WebSockets
| Path | Description |
|---|---|
| `/ws/dashboard` | Live host stats / VM state pushes for dashboards |
| `/ws/vm/{session_id}/activity` | Lightweight activity heartbeat channel |

## Response conventions

- Success responses return the resource directly (not wrapped) for
  single-object endpoints, or a JSON array for list endpoints.
- Errors follow FastAPI's default `{ "detail": "..." }` shape with an
  appropriate HTTP status code (401 unauthenticated, 403 forbidden, 404 not
  found, 409 conflict â€” e.g. "student already has an active VM").
